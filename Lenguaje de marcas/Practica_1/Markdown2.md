<h1 id="markdown">Markdown</h1>
<h2 id="informaci-n-y-comparaci-n-con-html">Información y comparación con HTML</h2>

<h2 id="introducci-n-">Introducción.</h2>
<p>Markdown es un lenguaje de marcado ligero diseñado para formatear texto de una manera que sea fácil de leer y escribir, y que al mismo tiempo pueda ser convertido a <em>HTML</em> u otros formatos de presentación.</p>
<p>Fue creado por <em>John Gruber</em> y <em>Aaron Swartz</em> en <em>2004</em> y se ha convertido en una herramienta esencial para la creación de documentación en muchos campos, incluyendo el desarrollo de software, la escritura técnica y la gestión de proyectos.</p>
<h2 id="beneficios-de-usar-markdown-">Beneficios de usar Markdown.</h2>
<p>Entre los beneficios de utilizar Makdown tenemos:</p>
<ol>
<li><p><strong>Simplicidad:</strong> Markdown es muy fácil de aprender y de usar. La mayoría de las personas pueden empezar a escribir en Markdown en cuestión de minutos, ya que utiliza una sintaxis muy sencilla y legible. </p>
</li>
<li><p><strong>Legibilidad:</strong> Los documentos escritos en Markdown son muy legibles incluso en su forma sin formato. Esto facilita la colaboración, ya que cualquiera puede revisar o editar documentos sin necesidad de herramientas específicas.</p>
</li>
<li><p><strong>Portabilidad:</strong> Como Markdown se puede convertir a HTML, PDF, Word y otros formatos, los documentos escritos en Markdown son altamente portátiles. Puedes generar versiones de tu documentación en diferentes formatos sin tener que volver a escribir todo.</p>
</li>
<li><p><strong>Amplio soporte:</strong> Markdown es ampliamente compatible con muchas herramientas y plataformas. Se utiliza en:</p>
</li>
<li>Sitios web</li>
<li>Wikis</li>
<li>Sistemas de control de versiones, </li>
<li><p>Editores de texto y más.</p>
</li>
<li><p><strong>Versionado de código:</strong> Muchas plataformas de desarrollo y sistemas de control de versiones como GitHub, GitLab y Bitbucket admiten Markdown en la documentación de proyectos, lo que permite integrar la documentación directamente en el flujo de trabajo de desarrollo.</p>
</li>
<li><p><strong>Personalización:</strong> Aunque Markdown es simple, también permite una cierta cantidad de personalización para ajustarse a tus necesidades específicas. Puedes agregar:</p>
</li>
<li>Tablas</li>
<li>Listas</li>
<li>Enlaces</li>
<li><p>Imágenes y más.</p>
</li>
